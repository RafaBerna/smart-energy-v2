\# Ingesta OMIE



\## Objetivo

Descargar precios OMIE, transformarlos y guardarlos en la base de datos.



\## Regla 1

Python solo ingiere datos.



\## Regla 2

No meter lógica de producto aquí.



\## Regla 3

La base de datos es la fuente central.



\## Regla 4

La web leerá de la base de datos, no del script.
